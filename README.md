WrittenBookTransfer
========================

软件介绍
--------

WrittenBookTransfer 是一款专为 Minecraft 玩家设计的成书数据库管理系统。该软件主要用于管理和组织 Minecraft 中的书面书籍（.mcfunction 文件），提供便捷的文件管理、搜索、翻译和AI分析功能，帮助玩家更好地整理和利用游戏中的书籍资源。

首先，请确保您已有 JAVA 25，运行src/main/java/WrittenBookTransferGUI.java就可打开初始化界面。

软件功能
--------

### 1. 数据库管理
- **打开数据库文件夹**：选择包含书籍文件的根目录，软件将自动扫描并加载所有 .mcfunction 文件。
- **目录结构浏览**：左侧树形结构显示文件夹层次，支持双击展开子目录。
- **书籍列表显示**：主数据库显示根目录下的书籍，子目录显示选中文件夹内的书籍。
- **实时刷新**：手动刷新数据库以同步最新文件变化。

### 2. 书籍操作
- **打开书籍**：双击书籍或右键选择"打开"查看书籍详情，包括标题、作者、页数、字数和内容。
- **拖拽移动**：支持将书籍拖拽到目录树或右侧面板的不同文件夹中，实现快速整理。
- **删除书籍**：右键选择"删除"将书籍移入回收站，避免误删。
- **移动书籍**：右键选择"移动到..."手动选择目标文件夹。
- **翻译功能**：右键选择"翻译为中文"将书籍内容翻译为中文。

### 3. 搜索功能
- **关键词搜索**：在搜索框输入关键词，支持空格分隔多个关键词。
- **搜索模式**：
  - 标题/作者：仅在书籍标题和作者中搜索
  - 全文检索：在书籍内容中进行全文搜索
- **实时过滤**：搜索结果实时显示在表格中。

### 4. AI 智能功能
- **AI 价值过滤**：使用 AI 分析书籍内容，过滤出有价值的书籍。
- **数据库预处理**：自动去重，清理重复的书籍文件。

### 5. 工具功能
- **回收站管理**：查看和管理已删除的书籍，支持恢复或永久删除。
- **存档提取成书**：从 Minecraft 存档中提取书面书籍到数据库。

### 6. 日志和状态
- **运行日志**：底部日志区域显示软件操作的详细记录。
- **状态栏**：显示当前操作状态和提示信息。

教程：如何使用 WrittenBookTransfer
------------------------------

### 步骤 1：启动软件
1. 双击运行 WrittenBookTransfer.jar 文件或通过命令行启动。
2. 软件窗口打开后，您会看到一个空的界面，状态栏显示"就绪"。

### 步骤 2：打开数据库文件夹
1. 点击菜单栏"文件" → "打开数据库文件夹"，或点击工具栏的"📂 打开文件夹"按钮。
2. 在弹出的文件选择对话框中，选择包含您的 Minecraft 书籍文件的根目录。
   - 通常是包含多个 .mcfunction 文件的文件夹。
3. 软件将自动扫描文件夹，左侧目录树会显示文件夹结构，主数据库表格会显示根目录下的书籍。

### 步骤 3：浏览书籍
1. 在左侧目录树中，双击文件夹可以展开查看子目录。
2. 选择一个子目录后，右侧"子目录"表格会显示该文件夹内的书籍。
3. 在表格中选择一本书籍，底部状态栏会显示基本信息。

### 步骤 4：查看书籍详情
1. 双击表格中的任意书籍，或右键点击书籍选择"📖 打开"。
2. 弹出一个新窗口显示书籍的详细信息：
   - 标题和作者
   - 页数和总字数
   - 完整内容（分页显示）

### 步骤 5：整理书籍
1. **拖拽移动**：
   - 在表格中选择一本书籍，按住鼠标左键拖拽。
   - 拖拽到左侧目录树的目标文件夹上释放。
   - 或拖拽到右侧子目录面板。
   - 软件会弹出确认对话框，点击"是"完成移动。

2. **右键菜单操作**：
   - 右键点击书籍打开上下文菜单。
   - 选择"📁 移动到..."手动选择目标文件夹。
   - 选择"🗑️ 删除"将书籍移入回收站。
   - 选择"🌐 翻译为中文"将书籍内容翻译。

### 步骤 6：搜索书籍
1. 在顶部搜索框输入关键词。
2. 选择搜索模式："标题/作者" 或 "全文检索"。
3. 点击"🔍 搜索"按钮或按回车键。
4. 表格会过滤显示匹配的书籍。

### 步骤 7：使用 AI 功能
1. 点击工具栏"🤖 AI 价值过滤"按钮。
2. 软件会使用 AI 分析所有书籍，过滤出有价值的书籍。
3. 点击菜单"工具" → "数据库预处理"进行去重处理。

### 步骤 8：管理回收站
1. 点击工具栏"🗑️ 回收站"按钮。
2. 在回收站管理窗口中：
   - 查看已删除的书籍
   - 选择书籍点击"恢复"移回原位置
   - 或点击"永久删除"彻底删除

### 步骤 9：提取存档书籍
1. 点击菜单"工具" → "📦 存档提取成书"。
2. 在弹出的工具中选择 Minecraft 存档文件。
3. 软件会自动提取存档中的书面书籍并添加到数据库。

### 步骤 10：退出软件
1. 点击菜单"文件" → "退出"，或直接关闭窗口。
2. 软件会自动保存当前状态并关闭。

常见问题解答
------------

**Q: 软件无法打开数据库文件夹？**
A: 确保选择的文件夹包含 .mcfunction 文件，且软件有读取权限。

**Q: 拖拽移动书籍时没有反应？**
A: 确保拖拽到有效的目标文件夹（目录树中的文件夹节点），且源文件存在。

**Q: AI 功能需要网络连接吗？**
A: 是的，AI 价值过滤和翻译功能需要互联网连接来调用 AI 服务。

**Q: 如何恢复误删的书籍？**
A: 使用回收站管理功能，可以恢复最近删除的书籍。

**Q: 软件支持哪些文件格式？**
A: 目前仅支持 Minecraft 的 .mcfunction 格式的书面书籍文件。

技术支持
--------

如果您在使用过程中遇到问题，请检查：
1. Java 运行环境是否正确安装（推荐 JDK 8 或更高版本）
2. 数据库文件夹权限设置
3. 网络连接（用于 AI 功能）
